# AWS CLI - Create IAM Role for EC2 Server Monitoring

This guide demonstrates how to create an IAM Role using the AWS CLI for the **AWS Server Monitoring & Log Backup System** project.

---

## Architecture

```
EC2 Instance
      │
      ▼
EC2-ServerMonitoring-Role
      │
      ├── AmazonSSMManagedInstanceCore
      ├── CloudWatchAgentServerPolicy
      ├── AmazonS3FullAccess
      └── AmazonElasticFileSystemClientReadWriteAccess
```

---

# Step 1: Create the Trust Policy

Create a file named **trust-policy.json**.

```bash
cat <<EOF > trust-policy.json
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Principal": {
                "Service": "ec2.amazonaws.com"
            },
            "Action": "sts:AssumeRole"
        }
    ]
}
EOF
```

---

# Step 2: Create the IAM Role

```bash
aws iam create-role \
    --role-name EC2-ServerMonitoring-Role \
    --assume-role-policy-document file://trust-policy.json
```

---

# Step 3: Attach Required Policies

## Attach Amazon SSM Managed Instance Policy

```bash
aws iam attach-role-policy \
    --role-name EC2-ServerMonitoring-Role \
    --policy-arn arn:aws:iam::aws:policy/AmazonSSMManagedInstanceCore
```

---

## Attach CloudWatch Agent Policy

```bash
aws iam attach-role-policy \
    --role-name EC2-ServerMonitoring-Role \
    --policy-arn arn:aws:iam::aws:policy/CloudWatchAgentServerPolicy
```

---

## Attach Amazon S3 Full Access Policy

```bash
aws iam attach-role-policy \
    --role-name EC2-ServerMonitoring-Role \
    --policy-arn arn:aws:iam::aws:policy/AmazonS3FullAccess
```

---

## Attach Amazon EFS Client Read/Write Policy

```bash
aws iam attach-role-policy \
    --role-name EC2-ServerMonitoring-Role \
    --policy-arn arn:aws:iam::aws:policy/AmazonElasticFileSystemClientReadWriteAccess
```

---

# Step 4: Create an Instance Profile

```bash
aws iam create-instance-profile \
    --instance-profile-name EC2-ServerMonitoring-Role
```

---

# Step 5: Add the Role to the Instance Profile

```bash
aws iam add-role-to-instance-profile \
    --instance-profile-name EC2-ServerMonitoring-Role \
    --role-name EC2-ServerMonitoring-Role
```

---

# Step 6: Verify the IAM Role

```bash
aws iam get-role \
    --role-name EC2-ServerMonitoring-Role
```

---

# Step 7: List Attached Policies

```bash
aws iam list-attached-role-policies \
    --role-name EC2-ServerMonitoring-Role
```

Expected Output

```
AmazonSSMManagedInstanceCore
CloudWatchAgentServerPolicy
AmazonS3FullAccess
AmazonElasticFileSystemClientReadWriteAccess
```

---

# Step 8: Verify the Instance Profile

```bash
aws iam get-instance-profile \
    --instance-profile-name EC2-ServerMonitoring-Role
```

---

# Step 9: Attach the IAM Role to an Existing EC2 Instance

Replace `<INSTANCE_ID>` with your EC2 instance ID.

```bash
aws ec2 associate-iam-instance-profile \
    --instance-id <INSTANCE_ID> \
    --iam-instance-profile Name=EC2-ServerMonitoring-Role
```

Example

```bash
aws ec2 associate-iam-instance-profile \
    --instance-id i-0123456789abcdef0 \
    --iam-instance-profile Name=EC2-ServerMonitoring-Role
```

---

# Verification

Verify that the IAM role has been successfully created.

```bash
aws iam get-role \
    --role-name EC2-ServerMonitoring-Role
```

Verify the attached managed policies.

```bash
aws iam list-attached-role-policies \
    --role-name EC2-ServerMonitoring-Role
```

Verify the instance profile.

```bash
aws iam get-instance-profile \
    --instance-profile-name EC2-ServerMonitoring-Role
```

---

# Cleanup

## Detach Policies

```bash
aws iam detach-role-policy \
    --role-name EC2-ServerMonitoring-Role \
    --policy-arn arn:aws:iam::aws:policy/AmazonSSMManagedInstanceCore

aws iam detach-role-policy \
    --role-name EC2-ServerMonitoring-Role \
    --policy-arn arn:aws:iam::aws:policy/CloudWatchAgentServerPolicy

aws iam detach-role-policy \
    --role-name EC2-ServerMonitoring-Role \
    --policy-arn arn:aws:iam::aws:policy/AmazonS3FullAccess

aws iam detach-role-policy \
    --role-name EC2-ServerMonitoring-Role \
    --policy-arn arn:aws:iam::aws:policy/AmazonElasticFileSystemClientReadWriteAccess
```

---

## Remove the Role from the Instance Profile

```bash
aws iam remove-role-from-instance-profile \
    --instance-profile-name EC2-ServerMonitoring-Role \
    --role-name EC2-ServerMonitoring-Role
```

---

## Delete the Instance Profile

```bash
aws iam delete-instance-profile \
    --instance-profile-name EC2-ServerMonitoring-Role
```

---

## Delete the IAM Role

```bash
aws iam delete-role \
    --role-name EC2-ServerMonitoring-Role
```
