# 🚧 Current Project

## AWS Project 1 – Server Monitoring & Log Backup System

### Objective

Build a production-style monitoring solution that automatically:

- Monitor EC2 server health
- Collect system metrics
- Store application logs
- Backup logs to Amazon S3
- Generate CloudWatch alarms
- Send SNS email notifications
- Record API activity using CloudTrail
- Secure resources using IAM
- Manage storage with EBS and EFS
- Create reusable Amazon Machine Images (AMI)

### AWS Services Used

- Amazon EC2
- IAM
- Amazon S3
- Amazon CloudWatch
- Amazon CloudTrail
- Amazon SNS
- Amazon EBS
- Amazon EFS
- Amazon AMI

### Skills Covered

- Linux Administration
- AWS IAM
- Monitoring
- Logging
- Alerting
- Storage
- Backup Strategy
- Security Best Practices
